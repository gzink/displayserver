DisplayServer (+ Client) for [SSD1803a](http://www.solomon-systech.com/en/product/display-ic/stn-lcd-driver-controller/ssd1803a/) lcd controller/EA DOGM204-A display using SPI

* written in python
* tested on Banana PI Pro
* [ZeroMQ](http://zeromq.org/) message broker

debian dependencies (wheezy): libzmq1 python-zmq
